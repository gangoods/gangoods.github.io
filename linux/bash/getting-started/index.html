    dd¶

    • 生成空文件 
      1
      dd if=/dev/zero of=test bs=10m count=1
    • 复制文件内容到另外一个文件 
      1
      dd if=abc.jpg of=test conv=notrunc
    • 跳过复制源多少字节 
      1
2
      # 跳过test,jpg的 10m内容, 后面的内容复制到my.mp4
dd if=test.jpg of=my.mp4 bs=10m skip=1
    • 跳过输出的目标文件的多少字节 
      1
2
3
      # 如果of指定的文件是不存在的,那么也会在文件前面写入空的10m 大小的数据
# 就是从指定文件的10m 大小位置开始写入if指定的文件
dd if=a.mp4  of=testbak.jpg  bs=10m seek=1

    () {}¶

    说明

    实现执行脚本时,没有参数直接提示错误, 然后退出脚本

    () 里实际是在子shell里执行命令,它的exit退出,变量修改等操作都不影响当前shell,所以不对

    {} 里面的是在当前shell的, 一组命令作为一个整体执行.

    错的版本
    1
2
3
    #!/bin/bash
test -z $1 && ( echo "need arg";exit; )
echo ok
    对的版本
    1
2
3
    #!/bin/bash
test -z $1 && { echo "need arg";exit; }
echo ok

    注意

    注意 需要{ } 两边空格, 最后 }前面的; 也是需要的
